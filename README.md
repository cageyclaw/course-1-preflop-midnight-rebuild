# Course 1 — Preflop (Static Course Site)

Simple static scaffold with sidebar navigation, previous/next controls, and responsive layout. Chapter content is loaded from markdown files.

## Content Location

Markdown source lives at:

```
company/projects/course-1-preflop/artifacts/course-md/*.md
```

These files are served as static assets and rendered client‑side.

## Run Locally

```bash
npm install
npm run dev
```

Open the URL printed by Vite (usually http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Notes

- Chapters A1–A7 are placeholders.
- Drills/quizzes intentionally **not** implemented.
