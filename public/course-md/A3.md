# Three‑Betting and Facing Pressure

## Objective
Build a 3‑bet plan that wins more pots preflop **and** avoids spewing when the heat comes back.

## Definitions
- **3‑bet:** Re‑raise versus an open.
- **Linear range:** Mostly strong hands (used when you expect calls).
- **Polar range:** Premiums + bluffs (used when you expect folds).

## Rules
1. **3‑bet for two reasons:** value + pressure.
2. **IP smaller, OOP bigger.** Default: **3x IP**, **3.5x OOP**.
3. **Choose linear vs polar** based on how often villain folds.
4. **No sizing tells.** Same size for bluffs and value.

![3-bet sizing quick guide](/course-md/3bet-sizing.svg)

## Exceptions
- **Call‑happy opponents:** skew linear and value‑heavy.
- **Over‑folders:** add more polar bluffs with blockers (A5s–A2s).
- **Squeeze spots:** size up when there are callers in between.
